# 手把手地教会第一批用户

做不可扩展的，必须手工做的事——这是 YC 创始人著名的说法。

Tony 提到的四个阶段
1. 自己
  - 团队自己能否成为重度用户 ？
  - 自己不是目标用户，是否也能为目标客户开发好作品？
2. 亲友
  - 亲友能否成为重度用户 ？
  - 亲友不能留存，在市场中推广中边走边看机会 ？机会很小
3. 早期外部用户
  - 团队自己是重度用户，部分亲友留存，可以进入这个阶段的试水
  - 能否吸引一批回头客
4. 口碑能否相传 ？
  - 回头客能否带来他的亲友 ？
  - 带动极少=未见到光，带动少许=少许希望，带动有力=见到光


一对一人肉推广，手把手地教会第一批用户

从家人、朋友、粉丝开始

例如：EatVenture 一对一邀请测试的例子
例如：知识星球早期邀请 Fenng、caoz 的例子

这是不可扩展的笨功夫，一定要下，并且做好记录

用户每一次拒绝，都是改进产品的机会
