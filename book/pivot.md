# 转型

## 什么是 Pivot

"Pivot"经常被创业提到，用来表示一种特定类型的转型：就像篮球中的"枢纽脚"（pivot），意味着在一只脚站稳不动的同时调整或改变方向。

【要点】
- 在进行 pivot 时，你只改变想法的一个维度
- 目的是发现那些如果同时改变多个维度就看不见的新洞见
- Lenny 数据：B2B 公司三分之一会 pivot，消费产品五分之一会 pivot

## 两种 Pivot 类型

【要点：Lenny 的关键框架】

1. Ideation Pivot（想法阶段转型）
   - 产品还没成型，没有真正用户
   - 换方向成本低，通常 3 个月内发生
   - 新想法往往和原来完全无关
   - 案例：Brex（VR 头盔 -> 企业信用卡）、Retool（英国版 Venmo -> 内部工具）

2. Hard Pivot（艰难转型）
   - 已有产品、真实用户
   - 保留一个元素，全力押注
   - 通常在 1-2 年发生
   - 案例：Instagram（签到 app -> 照片）、Slack（游戏 -> 聊天工具）

【数据】
- 四分之一的公司在发布后 3 个月内 pivot
- 中位数是 1 年
- Notion 用了 4 年，Lyft 用了 5 年

## 该 Pivot 的信号

【要点：两个核心信号】

信号一：持续的温吞反应
- 低留存 + 增长停滞 + 没有更好的增长点子
- Instagram："给人用，他们就是会弹开"
- Amplitude："留存很差，语音识别精度不够"
- Loom："7 个月只赚了 600 美元"

信号二：意识到天花板太低
- 随着经验积累，发现这个想法永远不会大到值得做
- Brex："回头看，这个 VR 想法真的很烂"
- Slack："Glitch 永远不可能成为一个值得 1720 万美元投资的生意"
- YouTube："整件事毫无意义，我们付钱让女人上传视频都没人来"

## 如何找到新方向

【要点：四种策略】

策略一：聚焦到一个有拉力的功能
- Loom：用户意外地爱上录屏功能
- Instagram：照片帖子的点赞和评论远超其他
- Yelp：用户自发开始写评论
- Pinterest：用户在 Tote 里做产品收藏
- Notion：wiki 和协作功能有人用

策略二：聚焦到一个有拉力的内部工具/技术
- Slack：做游戏时给自己做的聊天工具
- Amplitude：给自己做的分析工具，其他 YC 公司都想要
- Shopify：给自己做的电商后台，别人想 license
- Segment：Analytics.js 发到 Hacker News 直接上榜首
- Discord：游戏里的语音功能比游戏本身好用
- Hugging Face：周末 port 的 BERT 获得 1000 赞

策略三：转向更大的相邻市场
- Box：2-3% 付费转化太低，转向企业市场
- Coinbase：加了"购买比特币"按钮后才起飞
- Twitch：游戏直播只占 1-2%，但创始人第一次真正喜欢看自己的产品
- PayPal：15 个月内 pivot 了 5 次，最终聚焦 email 支付

策略四：内部头脑风暴
- Twitter：hackathon 中 Jack Dorsey 提出"状态"概念
- Lyft：hackathon 中移动团队想出全新设计
- Brex：YC 期间和 partner 讨论，找到"创业公司拿不到信用卡"的问题
- Vanta：不断问创业公司关于安全的问题，发现大家都知道该做但不做

## 坚持还是放弃

【要点：核心判断框架】

Scott Belsky 的问题：
"知道了现在知道的一切，你对这个解决方案的信念是增强了还是减弱了？"
- 如果增强了：你只是在 messy middle，坚持
- 如果减弱了：该放弃了，你的人生很短

Dalton Caldwell 的公式（伪代码）：
```
if ((进展程度) / (投入月数)) < (对新想法的兴奋度) * (找到更好想法的信心)
    -> 该 pivot 了
```

可以继续坚持的理由：
- 在做需要时间的复杂技术
- 有清晰的愿景，需要看到结果
- 玩得开心，不在乎商业结果

【Lenny 的最终建议】
"如果你现在读到这里，直觉告诉你该 pivot 了，那大概就是该 pivot 了。"
