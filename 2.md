# 第二稿考虑的修改

- 战略，组织是不是有涉及，是不是需要涉及？
- 原子能力和网络效应是不是跟留白有些关系，其实原子能力 + 网络效应，可能是个超级厉害的东西，微信群是这样，Manus 也是
- 考虑一下长期 = 长坡，厚雪 = 有复利，小落点 + 长坡 + 厚雪，或许机会就出来了
- 是不是足够酷，这个决定了能不能火
- 系统思考放到最后，会不会更好？


- 每一章自己读一遍，做出适合自己的修改，确保每句话都是自己想说的

- 每一章的开头，是不是可以都提出一个问题，这一章就是回答这一个问题的。
- 每一章的开头，引用一句“名言”，让人看了觉得有启发，并且与本章的内容、问题紧密关联
- 重新完整阅读这一章节，这一章的结构看起来合理吗？是不是遗漏了重要的部分？是不是有重复、冗余的结
构？是不是顺序做些调整，可以让整章更有说服力，更有感染力，更能打动人？
- 再仔细想深一点，想久一点，这真的是最好的结构吗？
- 这一章写得足够清晰吗？读者看完之后，能了解到的内容有哪些？
- 这一章有用讲故事、讲案例的技巧来帮助读者进一步深入理解了吗？是在说人话吗？是在讲常识吗？还是在讲一些大道理和空话套话？
- 这一章写得足够简洁吗？是不是有可以删掉的说教、偏离主题的内容、不恰当的类比或者案例等等。
