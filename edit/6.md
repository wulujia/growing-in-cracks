# ll 的建议

- [x] 关于支付系统

你在书里提了很多风控相关的内容，但我觉得【支付】才是最关键的。如果我们没有支付系统、后台系统和内容系统，尤其是支付系统，我觉得那些风控都是没有意义的。

星主们之所以能够忍受我们严格的风控，其实是因为我们有支付系统可以挣钱。

一款产品的成功，交易系统非常重要，我觉得这是一个事实。你在书里描述了很多，但是没有提到这一点：一款产品拥有交易系统，能够量化用户的努力是非常重要的。

比如“生财”做服务，其实做得很辛苦。因为他们没有交易系统，用户付的钱带来的具体收益其实是无法明确的。而有了交易系统，就意味着用户为我们花的钱是有明确结果的：

给知识星球的人头费带来了交易和收益，这种一比一的对比，会让风控变得不那么令人讨厌，同时我们的话语权也增加了。这也是一个很关键的细节。

星主们的推广虽然也带来了风险，但也保护了我们。他们推广的原因，就是因为用知识星球赚到了钱。

知识星球之所以能够赚钱，我们的预览页做得也非常出色。

说白了，整本书里都没有提到我们成功的关键：其实是因为在知识星球可以赚钱，它的收益是非常明确的。我觉得正因为能挣钱，才有了这款产品；它能挣钱的核心在于交易系统及其整体设计。

按我的理解，为什么之后我们好像就没有再找到新的突破？

星主用星球，赚钱变难了，我们收到的正反馈在下降，这是一个很赤裸的事实，书里好像完全忽略了这一点。你说了许多道理，但是没有说出这个事实，纯属个人感受。

所以我也想问你：你觉得你在坚守什么呢？你到底在坚守什么？


- [] 关于齐俊杰

齐老师对于很多星主来说是一座高山。有他的存在，大家会觉得做知识星球有一个对手，有一个王者。他的存在会激励很多人，甚至激励了亦仁。

齐老师多年如一日地保持更新。我认为齐俊杰这种极简的运营模式是非常有价值和意义的，也很简单。

生财有生财的牛逼，齐俊杰也有齐俊杰的伟大。
